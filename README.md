GERICHT — адаптивный одностраничный сайт ресторана в премиальном стиле. На сайте есть навигация по секциям (Home, About, Menu, Awards, Contact), главный hero-блок с CTA, блоки “About Us / Our History”, “Today’s Special” с меню напитков и коктейлей, “Chef’s Word”, “Our Laurels” (награды), фотогалерея, контакты и часы работы, форма подписки на рассылку и футер с соцсетями. Дизайн выполнен в тёмной теме с золотыми акцентами и крупной типографикой.
